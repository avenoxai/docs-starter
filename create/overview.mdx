## What Every Token Gets

Regardless of how you create it, every Slinq token includes:

| Feature | Details |
|---------|---------|
| **Fixed Supply** | 100 billion tokens (no inflation) |
| **Locked Liquidity** | LP locked permanently — no rugs possible |
| **Creator Rewards** | Up to 40% of all trading fees go to the creator |
| **MEV Protection** | Smart hooks protect against bots at launch |
| **Native DEX Trading** | Instant trading on PancakeSwap V4 — no bonding curve |
| **AI-Powered** | Names, tickers, and descriptions generated automatically for social launches |

## How Slinq Is Different

Unlike other launchpads that use bonding curves and then "migrate" to a DEX, Slinq tokens **trade natively on PancakeSwap from the very first trade**. This is possible because Slinq is built using PancakeSwap V4 **hooks** — smart contract plugins that run automatically on every trade to handle fees, MEV protection, and creator earnings.

## AI Agents Can Launch Tokens Too

Slinq has a dedicated API that allows AI agents and bots to create tokens programmatically. If you're building an AI agent, trading bot, or automated tool, you can:

- **Launch tokens** via API — no manual interaction needed
- **Get 1 free gas-sponsored launch per day** — the protocol covers the on-chain transaction cost
- **Set custom fee beneficiaries** — direct creator earnings to any wallet, Twitter handle, or Farcaster username
- **Trade tokens** — buy and sell through the API with a provisioned wallet

<Info>
AI agent tokens appear alongside all other tokens on Slinq. They have the same features — locked liquidity, creator fee sharing, and MEV protection through hooks.
</Info>
