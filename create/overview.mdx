## What Every Token Gets

Regardless of how you create it, every Slinq token includes:

| Feature | Details |
|---------|---------|
| **Fixed Supply** | 100 billion tokens (no inflation) |
| **Locked Liquidity** | LP locked permanently — no rugs possible |
| **Creator Rewards** | Creator revenue flows to the launch creator or beneficiary |
| **MEV Protection** | BNB Chain hook launches include launch protection against bots |
| **Native DEX Trading** | Ethereum launches use the Clanker wrapper; BNB launches trade on PancakeSwap |
| **AI-Powered** | Names, tickers, and descriptions generated automatically for social launches |

## How Slinq Is Different

Unlike launchpads that use bonding curves and then "migrate" to a DEX, Slinq tokens start on their real launch venue. Ethereum launches use Slinq's Clanker wrapper, while BNB Chain launches trade through PancakeSwap Infinity or V3. BNB Chain Infinity launches use PancakeSwap **hooks** to handle fees, launch protection, and creator earnings.

## AI Agents Can Launch Tokens Too

Slinq has a dedicated API that allows AI agents and bots to create tokens programmatically. If you're building an AI agent, trading bot, or automated tool, you can:

- **Launch tokens** via API — no manual interaction needed
- **Use payload mode for launches** — agents can prepare launches and let the user or caller execute when needed
- **Set custom fee beneficiaries** — direct creator earnings to any wallet, Twitter handle, or Farcaster username
- **Trade tokens** — buy and sell through the API with a provisioned wallet

<Info>
  AI agent tokens appear alongside all other tokens on Slinq. Ethereum launches use the Clanker wrapper; BNB Chain launches use Slinq's PancakeSwap stack.
</Info>

## Rate Limits

To maintain platform quality, each user has daily creation limits:

| Limit | Value |
|-------|-------|
| Total launch intents | 3 per requester per 24 hours |
| Sponsored BNB Chain launches | 1 per requester per 24 hours |
| Ethereum wrapper launches | Not gas-sponsored by Slinq; they wait for first buy or execution |

<Info>
Rate limits reset at midnight UTC. If you need higher limits for a legitimate project, contact us.
