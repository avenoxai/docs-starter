# Twitter Token Launch

Create a tradeable token by tweeting at `@slinqmeme`. Your token deploys in seconds and is immediately tradeable.

## Tweet Format

```
@slinqmeme deploy $TICKER "Token Name"
```

**Examples:**
```
@slinqmeme deploy $MOON "Moon Token"
@slinqmeme create $DOGE "Super Doge"
@slinqmeme launch $PEPE "Pepe 2.0"
```

<Tip>
Keywords `deploy`, `create`, and `launch` all work the same way.
</Tip>

## How It Works

<Steps>
  <Step title="Tweet your request">
    Post a public tweet mentioning `@slinqmeme` with your token ticker and name.
  </Step>
  <Step title="Bot processes your request">
    Slinq detects your mention and validates the parameters.
  </Step>
  <Step title="Token deploys">
    Your token is created on Ethereum through the Slinq Clanker wrapper by default, or on BNB Chain if you explicitly ask for BSC/BNB/PancakeSwap.
  </Step>
  <Step title="Receive confirmation">
    `@slinqmeme` replies to your tweet with:
    - Token contract address
    - Link to trade on Slinq
    - Your creator dashboard link
  </Step>
</Steps>

## Requirements

| Requirement | Details |
|------------|---------|
| **Twitter Account** | Must be public (not private/protected) |
| **Ticker** | 1-10 characters, letters and numbers only |
| **Name** | 1-50 characters |
| **Daily Limit** | 2 tokens per Twitter account |

## Token Parameters

Tokens created via Twitter use these defaults:

| Parameter | Value |
|-----------|-------|
| Default Chain | Ethereum |
| Ethereum Starting FDV | 2 ETH |
| BNB Chain Starting FDV | 10 BNB |
| Total Supply | 100 Billion |
| Trading Pair | TOKEN/WETH on Ethereum, TOKEN/WBNB on BNB Chain |
| Image | Custom upload or AI-generated |
| Description | AI-generated from your tweet |

<Info>
You can attach a custom image to your launch tweet, and the bot will use it as your token's image. Otherwise, one is AI-generated automatically.
</Info>

## Common Issues

<Accordion title="My tweet wasn't detected">
  - Make sure your account is public, not protected
  - Check that you mentioned `@slinqmeme` correctly
  - Verify your ticker doesn't contain special characters
  - Wait 2-3 minutes—there can be brief delays during high volume
</Accordion>

<Accordion title="Ticker already taken">
  Each ticker can only be used once. Try a variation:
  - Add numbers: `$MOON2`, `$DOGE99`
  - Use abbreviations: `$MTK` instead of `$MYTOKEN`
</Accordion>

<Accordion title="Rate limit reached">
  You can create 2 tokens per day via Twitter. Wait until midnight UTC for your limit to reset, or use the [web interface](/create/web) for additional launches.
</Accordion>

## Tokenize Someone Else's Tweet

You can also turn someone else's tweet into a token by replying:

```
@slinqmeme tokenize this
```

The **original tweet author** becomes the token creator and earns creator revenue. This is called [social tokenization](/create/social-tokenization).

<Tip>
Social tokenization keywords: tokenize, tokenise, make this a token, coin this, launch this, mint this
</Tip>
