## What Makes Slinq Different

Slinq is built directly into PancakeSwap V4 using **hooks** — smart contract plugins that run on every trade. This means:

- **No bonding curve** — Your token trades on a real DEX from the start
- **Dynamic fees** — The hook automatically adjusts fees to protect against bots and manipulation
- **Creator earnings** — The hook splits fees so creators earn from every single trade
- **Permanent liquidity** — LP is locked forever in a smart contract with no withdraw function
