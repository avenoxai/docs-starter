# 如何赚积分

Season Pass 积分主要有三种来源：交易、创作者收益和邀请。

## 交易积分

你在 Slinq 上每买卖一次代币，都会拿到积分。

| 行为 | 获得积分 |
|------|----------|
| 任意交易 | 每 1 BNB = 1 分 |
| 最低值 | 0.5 分（约等于 0.01 BNB 级别交易） |

### 计算方式

```
交易积分 = BNB 交易量 × 1 分 / BNB
```

**示例：**
- 买入 0.5 BNB 的 $MOON = 0.5 分
- 卖出 2 BNB 的 $DEGEN = 2 分
- 合计 = 2.5 分

<Info>
买入和卖出都一样计分。只要是在 Slinq 上的成交量，都会算进去。
</Info>

## 创作者收益积分

当你的代币产生交易手续费时，你还能额外拿到一笔积分。

| 行为 | 获得积分 |
|------|----------|
| 收到创作者手续费 | 每 1 BNB = 400 分 |
| 最低值 | 每次至少 2 分 |

### 计算方式

```
创作者积分 = 手续费收益（BNB）× 400 分 / BNB
```

**示例：**
- 你的代币累计交易量达到 100 BNB
- 1% 总手续费 = 1 BNB
- 你的 40% 分成 = 0.4 BNB
- 最终积分 = 0.4 × 400 = 160 分

<Tip>
如果你能做出一个持续有成交的币，涨分速度会远快于单纯自己刷交易。
</Tip>

## 邀请积分

你还可以从被你邀请的人身上拿积分分成。

| 行为 | 获得积分 |
|------|----------|
| 邀请奖励 | 被邀请人总积分的 25% |
| 月度上限 | 每个邀请人每月最多 100,000 分 |
| 持续时间 | 持续有效，不会过期 |

### 工作方式

当别人通过你的邀请码进入平台：
- 他们自己的所有积分会获得 10% bonus
- 你会拿到他们总积分的 25%

**示例：**
- 你邀请了 5 个朋友
- 每个人都赚了 1,000 分
- 你的邀请奖励 = 5 × 1,000 × 25% = 1,250 分

## 积分总结

| 来源 | 比例 | 备注 |
|------|------|------|
| 交易 | 1 分 / BNB | 买卖都算 |
| 创作者收益 | 400 分 / BNB | 来自你自己的代币 |
| 邀请 | 被邀请人的 25% | 受月度上限限制 |

## 等级倍率

更高等级会给你后续赚分加成：

| 等级 | 积分区间 | 倍率效果 |
|------|----------|----------|
| Bronze | 100-999 | 基础倍率 |
| Silver | 1K-9,999 | 1.2× - 1.3× |
| Gold | 10K-99,999 | 1.4× - 1.6× |
| Platinum | 100K+ | 2.0× - 2.2× |

<Info>
这些倍率只会作用在你达到该等级之后的新积分上。
</Info>

## 在哪里查看积分

排行榜页会显示你的积分拆分：

| 指标 | 说明 |
|------|------|
| **Creator Points** | 来自创作者收益的积分 |
| **Trader Points** | 来自交易的积分 |
| **Referral Points** | 来自邀请的积分 |
| **Total Points** | 三者总和 |
| **Current Rank** | 当前排行榜名次 |

## 常见问题

<Accordion title="积分会过期吗？">
  一个赛季内不会过期。新赛季开始时会统一重置。
</Accordion>

<Accordion title="有上限吗？">
  交易积分和创作者积分没有上限。邀请积分有每月 100,000 分上限。
</Accordion>

<Accordion title="积分多久更新一次？">
  通常是近实时更新。大多数活动后几分钟内你就能看到变化。
</Accordion>

<Accordion title="积分会倒扣吗？">
  不会。积分在赛季内只增不减。
</Accordion>
