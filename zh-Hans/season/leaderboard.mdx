# 排行榜

排行榜会按照 Season Pass 总积分对所有 Slinq 用户进行排序。排得越高，能拿到的奖励通常越多。

## 怎么看排行榜

1. 打开 [slinq.meme/leaderboard](https://slinq.meme/leaderboard)
2. 连接钱包后查看你的个人名次
3. 浏览头部用户和他们的总积分

## 排行榜里有什么

### Top 3 领奖台

前三名会获得更醒目的展示位：

| 名次 | 展示方式 |
|------|----------|
| 第 1 名 | 金色奖杯，最大展示 |
| 第 2 名 | 银牌展示 |
| 第 3 名 | 铜牌展示 |

### 排名表

领奖台下方会显示：

- 排名编号（#4、#5 等）
- 钱包地址简写
- 总积分
- 等级徽章（Bronze / Silver / Gold / Platinum）

### 你的数据卡

连接钱包后，你的个人卡片会显示：

| 指标 | 说明 |
|------|------|
| **Your Rank** | 当前名次 |
| **Total Points** | 总积分 |
| **Creator Points** | 来自创作者收益 |
| **Trader Points** | 来自交易量 |
| **Referral Points** | 来自邀请计划 |
| **Current Tier** | 当前等级 |
| **Progress** | 距离下一级还差多少分 |

## 等级体系

| 等级 | 所需积分 | 徽章 |
|------|----------|------|
| Bronze | 100 - 999 | 铜牌 |
| Silver | 1,000 - 9,999 | 银牌 |
| Gold | 10,000 - 99,999 | 金牌 |
| Platinum | 100,000+ | 钻石 |

<Info>
这些等级徽章不只会显示在排行榜，也会在平台其他位置展示你的状态。
</Info>

## 更新频率

| 数据 | 更新频率 |
|------|----------|
| 总积分 | 每 2 小时 |
| 排名 | 每 2 小时 |
| 你的个人数据 | 接近实时 |

<Tip>
个人数据通常比公开排行榜更新更快。如果你刚做完动作，先看个人页会更准。
</Tip>

## 最低显示门槛

想出现在公开排行榜里，需要满足：

- 最低 0.1 分
- 低于这个门槛的人不会出现在公共榜单
- 但你自己连接钱包后仍然能看到个人数据

## 赛季结束时

当赛季结束后：

<Steps>
  <Step title="最终快照">
    赛季结束时，积分会被锁定。
  </Step>
  <Step title="确认最终排名">
    系统会结算你的最终名次。
  </Step>
  <Step title="发放奖励">
    头部用户会收到链上奖励。
  </Step>
  <Step title="新赛季重开">
    积分归零，开始下一轮竞争。
  </Step>
</Steps>

## 常见问题

<Accordion title="排行榜多久更新一次？">
  公共排行榜每 2 小时更新一次。你的个人数据刷新会更快。
</Accordion>

<Accordion title="能看历史赛季排名吗？">
  当前赛季展示的是实时榜。过去赛季后续会以归档榜单的形式呈现。
</Accordion>

<Accordion title="为什么我总积分涨了，但排名变了？">
  排名是相对值。别人也在涨分，所以即使你在增加，总名次也可能上下波动。
</Accordion>

<Accordion title="头部用户能拿什么奖励？">
  奖励会按每个赛季单独公布。具体结构请以当季官方公告为准。
</Accordion>
