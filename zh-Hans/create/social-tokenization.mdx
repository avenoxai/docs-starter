# 社交代币化

看到一条正在爆的 tweet 或 cast？你只要回复一句关键词，就能把它变成可交易代币。原帖作者会自动成为创作者，并从后续每一笔交易里拿手续费。

## 怎么运作

<Steps>
  <Step title="发现一条值得发币的内容">
    找到一条你觉得应该被做成 token 的 tweet 或 Farcaster cast。
  </Step>
  <Step title="带关键词回复">
    在这条内容下面回复，并 @ `@slinqmeme`（Twitter）或 `@slinq`（Farcaster），同时带上触发关键词。
  </Step>
  <Step title="AI 自动生成代币">
    Slinq 的 AI 会读取原帖内容，自动生成适合的代币名称、Ticker 和描述。
  </Step>
  <Step title="代币上线">
    代币会以**原作者**作为创作者身份上线。后续每一笔交易手续费里，他们都能拿分成。
  </Step>
</Steps>

## 触发关键词

在任意帖子下回复下面这些关键词之一（同时记得 @ 机器人）：

```
tokenize, tokenise, make this a token, coin this, launch this, mint this,
turn this into a token, token this
```

**Twitter 示例：**
```
@slinqmeme tokenize this
```

**Farcaster 示例：**
```
@slinq coin this
```

<Tip>
这些关键词本身比较灵活，也支持多语言语境。只要回复里有触发短语，并且正确 @ 到机器人，系统就能识别。
</Tip>

## 谁会成为创作者？

这和普通发币最大的区别就在这里：

| 发币方式 | 谁是创作者 | 谁拿手续费 |
|----------|------------|------------|
| **社交代币化**（回复触发） | 原帖作者 | 原帖作者（40% 或 30%） |
| **普通发币**（主动请求） | 发起请求的人 | 发起请求的人（40%） |

<Info>
回复“tokenize this”的那个人**不是**创作者。真正的创作者是原帖作者。也就是说，你可以把一个 token 当成礼物送给别人，而后续收益归他们。
</Info>

## 代币参数

通过社交代币化生成的代币会使用这些设置：

| 参数 | 数值 |
|------|------|
| 初始 FDV | 5 WBNB |
| 总供应量 | 1000 亿 |
| 名称和 Ticker | AI 根据原帖内容自动生成 |
| 描述 | AI 自动总结原帖内容 |
| 图片 | 自动生成 |

<Info>
社交代币化默认使用更低的 5 WBNB 初始 FDV。相比普通发币更容易让人早期参与，也更贴合“热点即开盘”的场景。
</Info>

## 这些代币会出现在哪里？

这类由社交内容生成的代币会出现在 **Social Feed** 页面：[slinq.meme/socialfeed](https://slinq.meme/socialfeed)。它们会单独展示，不会和首页常规发币列表混在一起。

## AI 是怎么帮你发出来的

当你触发一次社交代币化时，Slinq 的 AI 会：

1. **读取原帖内容**，理解它在讲什么
2. **生成代币名称**，抓住帖子里的核心梗或情绪
3. **生成 Ticker**，尽量做到顺口、好记
4. **写出描述**，用帖子内容生成简短简介
5. **回复确认消息**，附上交易链接

你不需要自己手动填任何参数，AI 会帮你把这一套跑完。

## 手续费分成

当别人帮你把帖子代币化时，分成会这样走：

| 场景 | 创作者分成 | 首买者分成 | 协议 |
|------|------------|------------|------|
| 你自己把自己的帖子代币化 | 40% | — | 60% |
| 别人把你的帖子代币化 | 30% | 10% | 60% |

<Info>
如果是别人触发了你帖子对应代币的上线，他们会被视为“首买者”，并获得 10% 的手续费分成作为奖励。你作为原作者仍然会拿到 30%。
</Info>

## 常见问题

<Accordion title="如果原作者还没有钱包怎么办？">
  代币照样会被创建，手续费也会继续累积。等原作者之后把钱包接入 Slinq，就可以把之前累积的收益一次性提走。
</Accordion>

<Accordion title="我可以把自己的帖子代币化吗？">
  可以。你只要在自己的 tweet 或 cast 下回复触发关键词即可。这样你既是原作者，也是创作者。
</Accordion>

<Accordion title="如果这条帖子已经被代币化过了呢？">
  每一条帖子只能被代币化一次。如果已经有人抢先做了，系统会回复提示你。
</Accordion>

<Accordion title="原作者能拒绝这个代币吗？">
  代币本身是 permissionless 的，一旦上链就会永久存在。不过原作者可以选择不去领取手续费。
</Accordion>

<Accordion title="原作者需要做什么吗？">
  什么都不用做。收益会自动累积。等他们想提的时候，只需要把钱包连接到 Slinq 即可。
</Accordion>
