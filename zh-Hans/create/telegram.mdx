# Telegram 发币

你可以直接用 Slinq 的 Telegram 机器人来发币、追踪新盘、接收实时提醒，全程都不用离开 Telegram。

## 开始使用

<Steps>
  <Step title="找到 Slinq Bot">
    在 Telegram 搜索 `@SlinqBot`，或者直接加入我们的社区 [t.me/slinqxyz](https://t.me/slinqxyz)。
  </Step>
  <Step title="开始对话">
    给机器人发送 `/start`。
  </Step>
  <Step title="连接钱包">
    通过机器人里的 Web App 界面绑定你的钱包。
  </Step>
  <Step title="发币或交易">
    你可以直接在机器人里创建代币、浏览新盘和订阅提醒。
  </Step>
</Steps>

## 默认参数

通过 Telegram 创建的代币默认使用这些设置：

| 参数 | 数值 |
|------|------|
| 初始 FDV | 10 WBNB（可自定义） |
| 总供应量 | 1000 亿 |
| 交易对 | TOKEN/WBNB |
| 图片 | 自动生成 |
| 描述 | 自动生成 |

<Info>
如果你想要完整自定义，比如图片、描述和更细的 FDV 配置，建议直接使用 [Web 界面](/zh-Hans/create/web)。
</Info>

## Telegram Web App

Slinq 机器人内置了一个 Web App，相当于把完整平台做成了 Telegram 里的轻量版：

- 浏览热门代币
- 直接执行买卖
- 查看你的持仓
- 看排行榜名次

<Tip>
如果你经常在手机上操作，Telegram Web App 基本是最顺手的一种交易方式。
</Tip>

## 常见问题

<Accordion title="这个 Telegram 机器人是官方的吗？">
  是。官方机器人是 `@SlinqBot`。注意防伪，Slinq 不会主动私信你，也永远不会向你索要私钥。
</Accordion>

<Accordion title="我钱包里需要有 BNB 吗？">
  交易和 gas 都需要 BNB。发币本身仍然是 gasless 的，你只需要签一条消息。
</Accordion>

<Accordion title="Telegram 和 Twitter 可以一起用吗？">
  当然可以。代币和收益是绑定到你的钱包，不是绑定到入口平台。你可以在 Twitter 发币，在 Telegram 管理，在 Web 上交易，数据都是打通的。
</Accordion>
