# 开发者文档

Slinq 是基于 **PancakeSwap Infinity (V4)** 集中流动性 Hook 构建的代币发射平台，部署在 BNB Chain 上。本页涵盖了与 Slinq 链上基础设施集成所需的全部内容。

## 架构概览

Slinq 代币在 PancakeSwap V4 集中流动性池中原生交易。**Hook** 合约拦截每笔交换，处理动态手续费、MEV 保护和创作者手续费分配。没有联合曲线 - 代币从第一个区块起就是真正的 DEX 资产。

```
创作者签署 EIP-712 LaunchIntent（链下，无 Gas）
    |
首位买家调用 executeLazyLaunchAndBuy()
    |
LaunchExecutor: 验证签名 --> 工厂部署代币 (CREATE2) --> 初始化池 --> 添加流动性 --> 锁定 LP --> 注册到 Hook --> 执行首笔购买
    |
后续所有交易通过 SlinqRouter 进行
    |
Hook: 计算手续费（MEV 衰减 + 波动性）--> 分配（40% 创作者，60% 协议）
    |
LpLocker: 收割 LP 手续费 --> 销毁代币，BNB --> 回购并销毁 $SLINQ
```

## 集成交易

**SlinqRouter** 是交易的主要合约。它自动包装/解包原生 BNB，用户无需处理 WBNB。

### 购买代币

发送原生 BNB 购买任何 Slinq 代币：

```solidity
ISlinqRouter router = ISlinqRouter(0xd8c5711e640196C50D7fad60abBf9c464a60e97E);

// 用 0.1 BNB 购买代币
router.buyWithBNB{value: 0.1 ether}(
    tokenAddress,    // 要购买的 Slinq 代币
    amountOutMin,    // 最少接收代币数量（滑点保护）
    deadline         // 交易截止时间 (block.timestamp + 300)
);
```

### 卖出代币

先授权路由合约，然后卖出获取原生 BNB：

```solidity
// 1. 授权路由合约使用你的代币（一次性）
IERC20(tokenAddress).approve(address(router), type(uint256).max);

// 2. 卖出代币获取 BNB
router.sellForBNB(
    tokenAddress,    // 要卖出的 Slinq 代币
    amountIn,        // 卖出的代币数量 (uint128)
    amountOutMin,    // 最少接收 BNB 数量 (uint128)
    deadline         // 交易截止时间
);
```

<Warning>
Slinq 池仅支持 **精确输入** 交换。精确输出交换将会回滚。
</Warning>

### 获取价格报价

使用 **SlinqQuoter** 在交易前获取预期输出金额：

```solidity
ISlinqQuoter quoter = ISlinqQuoter(0xce45a9f0310B6dC60a7452CBec79D8f364621B80);

// 用 WBNB 购买代币的报价
(uint256 tokensOut, uint256 gas) = quoter.quoteBuyToken(
    tokenAddress,
    uint128(0.1 ether)  // WBNB 输入数量
);

// 卖出代币获取 WBNB 的报价
(uint256 bnbOut, uint256 gas) = quoter.quoteSellToken(
    tokenAddress,
    uint128(tokenAmount)  // 代币输入数量
);

// 通用报价（支持双向）
(uint256 amountOut, uint256 gas) = quoter.quoteExactInput(
    tokenAddress,
    amountIn,
    true    // true = 代币 --> WBNB（卖出），false = WBNB --> 代币（买入）
);
```

## 手续费结构

### Hook 手续费（交易手续费）

每笔交易收取 **1% 基础手续费**（最低），分配如下：

| 接收方 | 比例 |
|--------|------|
| 代币创作者 | 40% |
| Slinq 协议 | 60% |

如果代币由创作者以外的人发起（例如社交代币化），分配变为 30% 创作者 / 10% 首位买家 / 60% 协议。

### MEV 保护（发射溢价）

代币发射后的前 **120 秒**，买单需支付额外溢价，以抛物线形式衰减：

```
buyFee = 1% + 5% * (timeRemaining / 120)^2

t=0秒:   6%    (1% + 5%)
t=30秒:  ~3.8%
t=60秒:  ~2.25%
t=90秒:  ~1.3%
t=120秒: 1%    (溢价消失)
```

卖出手续费始终为 1%。MEV 溢价使抢跑无利可图。

### 波动性手续费

在价格高度波动期间，手续费使用 **VA 平方** 公式进行缩放：

```
volatilityPremium = (40000 * VA^2) / 1,000,000
totalFee = max(mevFee, baseFee + volatilityPremium)
```

最高总手续费**上限为 8%**。

### LP 手续费 (0.3%)

PancakeSwap 原生的 0.3% LP 手续费累积到锁定的流动性头寸。Slinq 收割这些手续费：
- **代币部分** 被永久销毁（通缩）
- **BNB 部分** 用于回购并销毁 $SLINQ

## 懒启动系统

Slinq 使用无 Gas 的"懒启动"模式。创作者免费创建代币 - 代币仅在首位买家出现时才会部署。

### 运作方式

<Steps>
  <Step title="创作者签署 LaunchIntent">
    一个 EIP-712 类型数据签名，包含代币名称、符号、图片 URI 和创作者地址。无需 Gas。
  </Step>
  <Step title="意图存储在链下">
    签署的意图存储在 Slinq 的后端，等待买家出现。
  </Step>
  <Step title="首位买家触发部署">
    买家在 LaunchExecutor 上调用 `executeLazyLaunchAndBuy()`，原子化地：部署代币 (CREATE2)、初始化池、添加流动性、锁定 LP，并执行首笔购买。
  </Step>
  <Step title="代币上线">
    后续所有交易通过 SlinqRouter 正常进行。
  </Step>
</Steps>

### LaunchExecutor 接口

```solidity
ISlinqLaunchExecutor executor = ISlinqLaunchExecutor(0xFB7a7F2815543d5c5821A526D15123765Ed2e100);

executor.executeLazyLaunchAndBuy{value: buyAmountInBNB}(
    intent,        // LaunchIntent 结构体（name, symbol, imageURI, creator, salt, deadline, fdv）
    creatorSig,    // 创作者的 EIP-712 签名
    amountOutMin,  // 首次购买的最少代币数量
    deadline,      // 交易截止时间
    buyerSig       // 买家授权签名
);
```

## 创作者手续费领取

创作者手续费在 **SlinqFeeLocker** 中累积，可随时领取。

```solidity
ISlinqFeeLocker feeLocker = ISlinqFeeLocker(0x89462054c284a051BD123d415EA0a2A93bbd2480);

// 领取单个代币的手续费
feeLocker.claim(creatorAddress, tokenAddress);

// 一次领取多个代币的手续费
address[] memory tokens = new address[](3);
tokens[0] = token1;
tokens[1] = token2;
tokens[2] = token3;
feeLocker.claimMultiple(creatorAddress, tokens);
```

<Info>
任何人都可以代替创作者调用 `claim()` - 手续费始终发送到创作者的钱包。这使得通过中继器进行无 Gas 领取成为可能。
</Info>

### 查询未领取的手续费

```solidity
// 读取未领取的手续费余额
uint256 unclaimed = feeLocker.feesToClaim(creatorAddress, tokenAddress);
```

## 池结构

所有 Slinq 池都是 PancakeSwap Infinity 上的 **集中流动性 (CL)** 池，具有以下属性：

- **交易对:** Token/WBNB（代币始终为 `currency0`）
- **LP 手续费:** 0.3% (3000)
- **Tick 间距:** 10
- **流动性:** 单边，全范围
- **LP 锁定:** 永久（无提取函数）
- **Hook:** SlinqHookVNext（动态手续费，MEV 保护）

### 读取池状态

```solidity
// 获取任何 Slinq 代币的池密钥

// 从 PancakeSwap 获取池状态
(uint160 sqrtPriceX96, int24 tick, , ) = CLPoolManager(poolManager).getSlot0(key.toId());
```

## 示例：完整购买流程 (ethers.js)

```javascript
import { ethers } from "ethers";

const ROUTER = "0xd8c5711e640196C50D7fad60abBf9c464a60e97E";
const QUOTER = "0xce45a9f0310B6dC60a7452CBec79D8f364621B80";

const provider = new ethers.JsonRpcProvider("https://bsc-dataseed.binance.org");
const signer = new ethers.Wallet(PRIVATE_KEY, provider);

const quoter = new ethers.Contract(QUOTER, [
  "function quoteBuyToken(address token, uint128 wbnbAmountIn) external returns (uint256 tokenAmountOut, uint256 gasEstimate)"
], provider);

const router = new ethers.Contract(ROUTER, [
  "function buyWithBNB(address token, uint128 amountOutMin, uint256 deadline) external payable returns (uint256 amountOut)"
], signer);

// 1. 获取报价
const bnbAmount = ethers.parseEther("0.1");
const [expectedOut] = await quoter.quoteBuyToken.staticCall(TOKEN, bnbAmount);

// 2. 应用 5% 滑点
const minOut = expectedOut * 95n / 100n;

// 3. 执行购买
const deadline = Math.floor(Date.now() / 1000) + 300;
const tx = await router.buyWithBNB(TOKEN, minOut, deadline, { value: bnbAmount });
await tx.wait();
```

## 示例：完整卖出流程 (ethers.js)

```javascript
const token = new ethers.Contract(TOKEN, [
  "function approve(address spender, uint256 amount) external returns (bool)",
  "function balanceOf(address account) external view returns (uint256)"
], signer);

const quoter = new ethers.Contract(QUOTER, [
  "function quoteSellToken(address token, uint128 tokenAmountIn) external returns (uint256 wbnbAmountOut, uint256 gasEstimate)"
], provider);

const router = new ethers.Contract(ROUTER, [
  "function sellForBNB(address token, uint128 amountIn, uint128 amountOutMin, uint256 deadline) external returns (uint256 amountOut)"
], signer);

// 1. 授权路由合约（一次性）
await (await token.approve(ROUTER, ethers.MaxUint256)).wait();

// 2. 获取余额和报价
const balance = await token.balanceOf(signer.address);
const [expectedBnb] = await quoter.quoteSellToken.staticCall(TOKEN, balance);

// 3. 应用 5% 滑点并卖出
const minBnb = expectedBnb * 95n / 100n;
const deadline = Math.floor(Date.now() / 1000) + 300;
const tx = await router.sellForBNB(TOKEN, balance, minBnb, deadline);
await tx.wait();
```

请始终通过 BSCScan 或我们的 `/api/config/contracts` 端点验证合约地址。切勿信任非官方来源的地址。
</Warning>
